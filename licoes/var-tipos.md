## 🎯 Resumo Executivo
Variáveis são contêineres para armazenar valores de dados. Em Python, a tipagem é dinâmica e forte, o que significa que você não precisa declarar o tipo explicitamente, mas o interpretador garante a integridade das operações entre tipos compatíveis.

## 📚 Conteúdo Detalhado

### 1. Tipos Primitivos em Python
- **Integers (int):** Números inteiros (ex: `10`, `-5`).
- **Floats (float):** Números decimais (ex: `10.5`, `3.14`).
- **Strings (str):** Sequências de caracteres (ex: `"Olá Mundo"`).
- **Booleans (bool):** Valores lógicos (`True` ou `False`).

### 2. Declaração e Atribuição
- **Conceito Chave:** O operador `=` é usado para atribuir valores a variáveis.
- **Nomenclatura:** Use `snake_case` (ex: `minha_variavel`).
- **Exemplo:**
  ```python
  idade = 25          # int
  preco = 19.99       # float
  nome = "Lucinei"    # str
  ativo = True        # bool
  ```

## 💡 Insights e Conexões
- **Por que importa:** Entender os tipos de dados é crucial para evitar erros de execução e para gerenciar a memória de forma eficiente.
- **Conexões:** Conecta-se diretamente com **Operadores** (que manipulam esses dados).
- **Limitações:** Python não possui constantes nativas imutáveis (usa-se convenção de MAIÚSCULAS).
