## 🎯 Resumo Executivo
A Lógica de Programação é a base técnica necessária para desenvolver o raciocínio estruturado. Ela envolve a capacidade de decompor problemas complexos em passos simples que um computador possa executar.

## 📚 Conteúdo Detalhado

### 1. Pensamento Computacional
- **Conceito Chave:** Decomposição, Reconhecimento de Padrões, Abstração e Algoritmos.
- **Mecanismo:** Transformar uma necessidade humana em uma sequência lógica de instruções.

### 2. Estruturas de Controle
- **Sequencial:** Execução linha a linha.
- **Condicional:** Tomada de decisão baseada em valores booleanos (`if`, `else`).
- **Repetição:** Execução de blocos múltiplas vezes até que uma condição seja satisfeita (`loops`).

## 💡 Insights e Conexões
- **Por que importa:** Uma boa lógica permite que você aprenda qualquer linguagem de programação em poucos dias, pois as estruturas fundamentais são as mesmas.
- **Conexões:** É o pré-requisito absoluto para **Sintaxe e Estruturas** e **Algoritmos**.
- **Limitações:** A lógica pura não resolve problemas de performance em larga escala sem o conhecimento de Estruturas de Dados avançadas.
