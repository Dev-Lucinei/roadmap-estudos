# Roadmap Orchestrator

Orquestrador de roadmaps de estudo com diagnóstico, geração e renderização.

## Arquitetura

- **Backend**: `server.py` — API REST na porta 8000
- **Frontend**: `index.html`, `app.js` — Renderização interativa de roadmaps
- **Geração IA**: `generate_roadmap.py`, `generate_lessons.py` — Integração OpenRouter
- **Dados**: `/data/*.json` — Roadmaps armazenados; `/licoes/*.md` — Lições geradas
- **Persistência**: `dep_map.json` — Mapa de dependências entre tópicos

## Endpoints

- `GET /api/roadmaps` — Lista todos os roadmaps disponíveis
- `GET /api/roadmap/<nome>` — Retorna roadmap específico
- `POST /api/diagnose` — Diagnóstico de lacunas de conhecimento
- `POST /api/generate` — Gera novo roadmap via IA

## Comandos

```bash
uv run python3 server.py          # Iniciar servidor
uv run python3 generate_roadmap.py <tema>   # Gerar roadmap
uv run python3 generate_lessons.py <topico> # Gerar lição
```
