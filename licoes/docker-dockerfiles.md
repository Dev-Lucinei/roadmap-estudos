# Dockerfiles

Aprenda a criar imagens Docker personalizadas usando Dockerfiles.

## O que é um Dockerfile?

Um Dockerfile é um arquivo de texto que contém instruções para construir uma imagem Docker automaticamente.

## Instruções Básicas

- `FROM`: Define a imagem base
- `RUN`: Executa comandos durante o build
- `COPY`: Copia arquivos para a imagem
- `CMD`: Define o comando padrão ao iniciar o container
